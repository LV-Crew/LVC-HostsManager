LV-Crew HostsManager


This program allows you to update your hosts file with a downloaded blacklist.
The blacklisted traffic is being redirected to a blockpage (or a URL/localhost).
Therefore tracking is being reduced to a minimum.
You are also being protected from spam ads and more.
(In short...)

And now in long... ;-)


- The Hosts-File-Downloader & -Updater pulls a new/updated hosts-file from any online source (URL).
- The Hosts-Files-List is saved by the app.
More to the hosts-file: https://en.wikipedia.org/wiki/Hosts_(file)


Antivirus





Many antivirus programs do block the hosts file in a way, that we can't unblock it.
Please refer to your antivirus program's manual to find out how to disable hosts file protection.

Avira Antivirus: https://answers.avira.com/de/question/avira-blocks-hosts-file-what-can-i-do-90
Others: https://www.devside.net/wamp-server/unlock-and-unblock-the-windows-hosts-file

Main






Select one or more blacklists to update your system.
The selected blacklists will get added into your hosts file.


Tools






Edit hosts file: Edit the hosts file.
Remove duplicates: Removes duplicate entrys from the hosts file.
Reset hosts file: Set hostsfile to Windows standard.
Flush DNS cache: removes temporarily saved DNS data.
Disable DNS Service: Do this if your System slows down while surfing.
Set DNS Server to Google: Set your DNS Server IP to that of Google (8.8.8.8).
Set DNS Server to OpenDNS: Set your DNS Server IP to that of OpenDNS.


Settings - Main Options






Redirection: Sets the IP to which blacklisted hosts are being redirected to.
Hosts file editor: Sets the editor that is being used by Hosts Manager to edit the hosts file.
Automatically update host file: Creates a Windows task which updates the hosts file daily.

Overwrite hosts file URLs:
=================================================================
This allows you to choose the URLs of the hosts files to be imported manually, overwriting the system setting.

Overwrite IP:
=================================================================
Most host file are in the format 0.0.0.0 , which redirects the traffic to localhost.
LV-Crew HostsManager allows you to overwrite this address with an address you choose.
By default, it redirects to a white page.

Hosts file editor:
=================================================================
Choose whether to use the internal text editor or wordpad for editing the hosts file.

Automatically update hosts file hourly:
=================================================================
This allows you to automatically update the hosts file.
It creates a new task in windows' task planer.
